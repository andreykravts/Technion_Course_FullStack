Font Properties

We can style any text using text CSS properties. Text font properties are described here. See CSS Code window for external styling applied to this page. Feel free to change it. Look also at CSS Text Generator to generate CSS code for your text. Look also Font Properties Cheat Sheet.

font-family - There are a few "safe" fonts (the most commonly used are Arial, Helvetica, Verdana and Times New Roman), but you can specify more than one font, separated by commas. See A complete collection of web safe CSS font stacks. See also Commonly Used Font Combinations.

font-size - Sets the size of the font. You can specify the font size in any of the standard CSS units.

font-weight - Sets the thickness of text. Most commonly used values are bold and normal.

font-style - Sets the font as italic or normal.

font-variant - Specifies if a text should be in a small-caps.

Advanced feature: the @font-face CSS at-rule allows to specify custom font with which to display text; the font can be loaded from either a remote server or the user's own computer.

Useful Resources

You can find great fonts at Google Fonts web site. It is free resource, where you can get web fonts. You can include them directly without having to download any files. My favorite is Montserrat font for headers.

The future of web fonts is variable font. See for example Variable Fonts or Another Variable Fonts. It is a new technology (check in Can I Use who supports it).

There are also layered fonts, which can be used for special effects. See for example Core Circus Font.

For special symbols font (font icons) you have a few possibilities. See What are the best icon web fonts? and use it in your design. See Font Awesome examples here. You can find a good explanation about Font Awesome and Google Icons in w3schools site here.
