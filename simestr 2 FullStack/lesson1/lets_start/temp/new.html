Try Your Styles

Write your code in the HTML Code window and define external, internal and inline styles. Write external styles in the CSS Code window. We assume that the code in the CSS window is actually the code written in external file (styles/my-css.css), which is linked in the head element.

For example:

  1. Define <p> element as navy in external CSS Code window. Check the result.
  2. Add internal style with different color (e.g green) for <p>. Check the result.
  3. Finally, add inline style style with different color from external and inline (e.g. red) for <p> and check.

Remember: Inline style has higher priority, so it overwrites internal and external style.
